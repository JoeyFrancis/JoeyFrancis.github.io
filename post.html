Sports Team Website

This was a project I started in university with other 2 coleagues and we used django and some JS.

This website have a shop, a calendar with the team events, a login page and much more!

In the backend we used django as our database and use JS, HTML and CSS in the front-end.

If you have an account you can buy items from the shop and tickets for the games.

This was a 2 months project and i learned a lot while working on it.

Technologies:
  • - Python
  • - Django
  • - JS
  • - HTML/CSS
